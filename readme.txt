Display the count of all types of comments.

== Description ==
This plugin will help you to display the number of comments in different categories and adds a widget to the WordPress dashboard.

== Installation Process ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "User Counter"
3. **Install and Activate** User Counter from your Plugins page
4. Bingo, your plugin has been activated, and your widget is now in your WordPress Dashboard.
